PIAIC offers the following cources:

  • Artificial Intellegence
  • Cloud Native Computing
  • Blockchain Computong
  • Internet of Things

Artificial Intellegence

The Artificial Intellegence program of th PIAIC takes the student through the foundation of t he AI technology, via teaching programing in open source languages like Python, Numpy, Pandas, Tenseflow 2.0 and Flask, and teaching developing AI solutions and use cases, and Ending with real-life examples of modern day AI in practice.








Cloud Native Computing

The Cloud Native Computing program prepare the student to program in open source technologies like Lunix, Docker, Kubernetes, JavaScript, Node,js, React and React Native while introducing students to development od microservices and software development techniques on cloud platforms like AWS.








Blockchain Computing

The Main Emphasis of Blockchain is to understand Blockchain like Ethereum and Hyperledger (Fabric) and learn to program in laguages such as Solidity, JavaScript on framework like Node.js. The Program also introduces them to development of Distributed Apps (DApps), frameworks like web3 and truffle and iin-depth understanding of Blockchain-as-a-service (Baas) and how to develop and offer an Initial Coin Offering (ICO) of a Crypto-token.








Internet of Things

The Internet of Things program intoduces th students to open source development technologies like Rust, Embedded Rust, WebAssembly, Hyper, Diesel, Google Asistant, and kubernetes while making them efficient in developing Microservice for IOT edge, IOT Cloud, and voice computing for device control.
